SISTEMA DE GESTIÓN DE LICENCIAS DE CONDUCIR

.

Este Proyecto es una apliación de escritorio desarrollado en Java para la automatización de generación y emisión de licencias de Conducir. El sistema hace desde la gestión de Usuarios hasta el registro, validación, evaluación y emisión de una licencia digital.

.

TECNOLOGIAS QUE SE UTILIZARON

Lenguaje: Java 21 

Interfaz: Java Swing (Diseño Híbrido & MVC)

Base de Datos: PostgreSQL 15 (Cloud - Supabase)

Seguridad: jBCrypt (Hashing) + RBAC (Roles)

Reportes: iText PDF

Construcción: Gradle

.

FUNCIONALIDADES PRINCIPALES

Seguridad Robusta: Login encriptado y control de acceso por roles (Administrador y Analista).

Gestión de Conductores: Validación de cédula ecuatoriana.

Pruebas Psicométricas: Cálculo automático de promedios y aprobación.

Validación de Requisitos: Checklist administrativo para documentos físicos.

Emisión de Licencias: Generación de PDF oficial con vigencia automática de 5 años.

Cloud Native: Persistencia de datos en la nube (Supabase)

.
